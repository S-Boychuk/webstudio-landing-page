# 🏢 WebStudio Corporate Landing Page

A modern and fully responsive corporate landing page for a web development studio. Built to practice semantic markup, responsive layout techniques, and UI component testing.

---

## 🌐 Live Demo
Experience the live project here: **[Live Page Link](https://s-boychuk.github.io/webstudio-landing-page/)**

---

## 🛠️ Tech Stack & Key Features
* **Technologies:** HTML5, CSS3 (Custom Properties, Flexbox, Transitions), SVG icons.
* **Responsive Design:** Fully adaptive layout tested across Mobile (320px+), Tablet (768px+), and Desktop (1158px+) viewports.
* **UI Components:** Styled modal windows, mobile navigation menu, and interactive subscription/contact forms.

---

## 🔍 QA & Layout Verification (Manual Testing)
* **Cross-Browser Testing:** Verified correct visual rendering on Chrome, Firefox, and Safari.
* **Responsive Validation:** Checked layout stability and element scaling across different screen sizes.
* **Form Testing:** Verified placeholder states, focus management, and basic input behavior.
